자바스크립트의 비동기 처리는 싱글 스레드 환경에서 효율적으로 작업을 처리하기 위한 핵심 개념이다.

## Promise

ES6에서 도입된 Promise는 비동기 작업의 결과를 나타낸다.

```javascript
const fetchData = () => {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      resolve('데이터 로드 완료');
    }, 1000);
  });
};
```

## async/await

async/await는 Promise를 더 직관적으로 사용할 수 있게 해준다.

```javascript
const loadData = async () => {
  try {
    const data = await fetchData();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
};
```

비동기 처리를 제대로 이해하면 웹 개발의 많은 부분이 명확해진다.

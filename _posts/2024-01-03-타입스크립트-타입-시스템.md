타입스크립트의 타입 시스템을 이해하면 더 안전한 코드를 작성할 수 있다.

## 기본 타입

```typescript
let name: string = "hello";
let age: number = 25;
let isActive: boolean = true;
```

## 인터페이스

객체의 구조를 정의할 때 인터페이스를 사용한다.

```typescript
interface User {
  name: string;
  age: number;
  email?: string; // 선택적 속성
}
```

## 제네릭

재사용 가능한 타입을 만들 때 제네릭을 활용한다.

```typescript
function identity<T>(arg: T): T {
  return arg;
}
```

타입 안정성을 확보하면서도 유연한 코드를 작성할 수 있다.

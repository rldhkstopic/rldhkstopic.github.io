## 제목 스타일

### H2 제목 (섹션 제목)

이것은 H2 제목입니다. 주요 섹션을 구분할 때 사용됩니다. 제목 아래에는 본문 내용이 이어집니다.

#### H3 제목 (하위 섹션)

H3 제목은 H2보다 작은 하위 섹션을 나타냅니다. 여러 단계의 계층 구조를 만들 때 유용합니다.

##### H4 제목 (세부 항목)

H4 제목은 더 작은 세부 항목을 나타냅니다. 깊은 계층 구조가 필요할 때 사용할 수 있습니다.

## 본문 텍스트 스타일

이것은 일반적인 본문 텍스트입니다. 블로그 포스트의 주요 내용은 이 스타일로 작성됩니다. 충분히 긴 텍스트를 작성하여 가독성과 줄 간격을 확인할 수 있습니다. 여러 문장이 이어지면서 자연스러운 흐름을 만들어냅니다.

**강조 텍스트 (Bold)**는 중요한 내용을 강조할 때 사용됩니다. 예를 들어, 중요한 개념이나 핵심 키워드를 강조할 수 있습니다.

*이탤릭 텍스트 (Italic)*는 부가적인 설명이나 강조를 위해 사용됩니다. 예를 들어, 특정 용어의 정의나 참고사항을 표시할 때 유용합니다.

***굵은 이탤릭 텍스트***는 강조와 이탤릭을 동시에 사용하는 경우입니다.

## 리스트 스타일

### 순서 없는 리스트 (Unordered List)

- 첫 번째 항목입니다. 각 항목은 불릿 포인트로 표시됩니다.
- 두 번째 항목입니다. 리스트는 여러 줄에 걸쳐 작성할 수 있습니다.
  - 중첩된 항목도 가능합니다. 들여쓰기를 통해 하위 항목을 만들 수 있습니다.
  - 또 다른 중첩 항목입니다.
- 세 번째 항목입니다. 리스트는 계속 이어질 수 있습니다.

### 순서 있는 리스트 (Ordered List)

1. 첫 번째 순서 항목입니다. 번호가 자동으로 매겨집니다.
2. 두 번째 항목입니다. 순서가 중요한 내용을 나열할 때 사용합니다.
3. 세 번째 항목입니다.
   1. 중첩된 순서 리스트도 가능합니다.
   2. 하위 단계를 나타낼 수 있습니다.
4. 네 번째 항목입니다.

## 코드 스타일

### 인라인 코드

인라인 코드는 `const variable = "value"`와 같이 문장 중간에 코드를 삽입할 때 사용됩니다. 예를 들어, `function name()`이나 `array.length` 같은 짧은 코드 조각을 표시합니다.

### 코드 블록

코드 블록은 여러 줄의 코드를 표시할 때 사용됩니다. 예를 들어:

```javascript
function calculateSum(numbers) {
  let sum = 0;
  for (let i = 0; i < numbers.length; i++) {
    sum += numbers[i];
  }
  return sum;
}

const result = calculateSum([1, 2, 3, 4, 5]);
console.log(result); // 15
```

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# 피보나치 수열의 10번째 항
result = fibonacci(10)
print(f"10번째 피보나치 수: {result}")
```

## 코드와 설명의 조합

프로그래밍 관련 포스트에서는 코드와 설명이 함께 사용되는 경우가 많습니다.

다음은 JavaScript의 배열 메서드를 사용하는 예시입니다:

```javascript
const numbers = [1, 2, 3, 4, 5];

// map을 사용하여 각 요소를 제곱
const squared = numbers.map(n => n * n);
console.log(squared); // [1, 4, 9, 16, 25]

// filter를 사용하여 짝수만 필터링
const evens = numbers.filter(n => n % 2 === 0);
console.log(evens); // [2, 4]

// reduce를 사용하여 합계 계산
const sum = numbers.reduce((acc, n) => acc + n, 0);
console.log(sum); // 15
```

이 코드는 배열의 다양한 메서드를 보여줍니다. 각 메서드는 배열을 다루는 강력한 도구입니다.

## 긴 코드 블록 예시

복잡한 알고리즘이나 긴 코드 예시를 보여줄 때는 다음과 같이 작성할 수 있습니다:

```python
class BlogPost:
    def __init__(self, title, content, author, date):
        self.title = title
        self.content = content
        self.author = author
        self.date = date
        self.views = 0
        self.tags = []
    
    def add_tag(self, tag):
        """태그를 추가합니다."""
        if tag not in self.tags:
            self.tags.append(tag)
    
    def increment_views(self):
        """조회수를 증가시킵니다."""
        self.views += 1
    
    def get_summary(self, length=100):
        """포스트의 요약을 반환합니다."""
        if len(self.content) <= length:
            return self.content
        return self.content[:length] + "..."
    
    def __str__(self):
        return f"제목: {self.title}\n작성자: {self.author}\n날짜: {self.date}\n조회수: {self.views}"

# 사용 예시
post = BlogPost(
    title="파이썬 리스트 컴프리헨션 활용법",
    content="리스트 컴프리헨션은 파이썬에서 리스트를 간결하게 생성하는 방법입니다...",
    author="kiwanPark",
    date="2024-01-20"
)

post.add_tag("Python")
post.add_tag("프로그래밍")
post.increment_views()

print(post)
print(f"요약: {post.get_summary()}")
```

이 코드는 블로그 포스트를 나타내는 클래스를 정의하고, 다양한 메서드를 통해 포스트를 관리하는 방법을 보여줍니다.

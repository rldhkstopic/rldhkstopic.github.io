리스트 컴프리헨션은 파이썬에서 리스트를 간결하게 생성하는 방법이다. 반복문과 조건문을 한 줄로 표현할 수 있어 코드가 훨씬 간결해진다.

## 기본 문법

```python
# 일반적인 방법
numbers = []
for i in range(10):
    numbers.append(i * 2)

# 리스트 컴프리헨션
numbers = [i * 2 for i in range(10)]
```

## 조건문 활용

필터링이 필요한 경우 조건문을 추가할 수 있다.

```python
# 짝수만 필터링
even_numbers = [i for i in range(20) if i % 2 == 0]
```

## 중첩 반복문

여러 개의 반복문을 중첩해서 사용할 수도 있다.

```python
matrix = [[i * j for j in range(3)] for i in range(3)]
```

리스트 컴프리헨션을 적절히 활용하면 코드의 가독성과 효율성을 모두 높일 수 있다.

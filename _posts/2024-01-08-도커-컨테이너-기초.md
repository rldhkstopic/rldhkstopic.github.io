도커를 처음 사용하면서 배운 기본 개념들을 정리했다.

## 컨테이너란

컨테이너는 애플리케이션과 그 실행 환경을 함께 패키징한 것이다. 가상머신보다 가볍고 빠르게 실행할 수 있다.

## 기본 명령어

```bash
# 이미지 빌드
docker build -t myapp .

# 컨테이너 실행
docker run -d -p 8080:80 myapp

# 실행 중인 컨테이너 확인
docker ps
```

## Dockerfile 작성

애플리케이션을 컨테이너화하기 위해 Dockerfile을 작성한다.

```dockerfile
FROM node:18
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["node", "index.js"]
```

도커를 사용하면 개발 환경과 프로덕션 환경의 차이를 최소화할 수 있다.

개발하면서 자주 사용하는 리눅스 명령어들을 정리했다.

## 파일 관리

```bash
# 파일 목록
ls -la

# 파일 복사
cp source.txt dest.txt

# 파일 이동
mv old.txt new.txt

# 파일 삭제
rm file.txt
```

## 프로세스 관리

```bash
# 실행 중인 프로세스 확인
ps aux

# 프로세스 종료
kill -9 PID
```

## 권한 관리

```bash
# 권한 변경
chmod 755 script.sh

# 소유자 변경
chown user:group file.txt
```

터미널에서 작업할 때 유용한 명령어들이다.

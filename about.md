# 소개

이 블로그는 AI와의 대화를 정리하고 지식을 보존하기 위해 만들어졌습니다.

## 목적

- AI와의 대화 내용을 체계적으로 정리
- 학습한 내용과 해결한 문제들을 문서화
- 지식의 축적과 공유

## 사용 방법

1. `_posts/` 폴더에 새로운 마크다운 파일을 생성합니다
2. 파일명 형식: `YYYY-MM-DD-제목.md`
3. Front Matter에 제목, 날짜, 태그 등을 작성합니다
4. 마크다운 형식으로 내용을 작성합니다
5. GitHub에 푸시하면 자동으로 블로그에 반영됩니다

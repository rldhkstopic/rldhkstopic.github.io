## 텍스트 스타일

**굵은 텍스트**: 중요한 내용을 강조할 때 사용합니다.

*기울임 텍스트*: 특별한 의미나 강조가 필요한 부분에 사용합니다.

***굵은 기울임 텍스트***: 매우 중요한 내용을 강조할 때 사용합니다.

~~취소선 텍스트~~: 삭제되었거나 더 이상 유효하지 않은 내용을 표시할 때 사용합니다.

# 블로그 변경 내역

이 파일은 블로그의 기능 추가, 레이아웃 변경, 스타일 수정 등 주요 변경 사항을 기록합니다.

## [v0.0.3] - 2026-01-08

### Changed
- 모바일/태블릿 홈 상단 프로필 바를 2줄 레이아웃으로 재구성(프로필 1행, 상태/오늘 방문자 2행)
- 상태메시지 톤을 회색/작은 크기로 조정
- 오늘 방문자수 표시를 텍스트 형태로 단순화

## [v0.0.5] - 2026-01-08

### Changed
- 모바일에서 포스트 제목 크기 축소 및 줄바꿈 개선(폭에 맞춘 가독성)
- 본문 헤딩(h2/h3) 하단 border 제거, 타이포로만 구분되도록 조정

## [v0.0.6] - 2026-01-08

### Fixed
- 코드 블럭 내부 스크롤 시 라인넘버가 함께 스크롤되지 않던 문제 수정(스크롤 컨테이너 일원화)

## [v0.0.7] - 2026-01-08

### Fixed
- 긴 코드 블럭에서 스크롤이 사라지던 문제 수정(`code-expandable`의 overflow-y 정책 조정)

## [v0.0.8] - 2026-01-08

### Fixed
- 코드 블럭 라인넘버 줄 간격이 과도하게 좁아 보이던 문제 개선(라인높이 변수로 통일)

## [v0.0.9] - 2026-01-08

### Fixed
- 코드 블럭 라인높이 적용이 테마 CSS에 의해 체감되지 않던 문제 보완(우선순위 강화 및 간격 상향)

## [v0.0.10] - 2026-01-08

### Added
- VHDL 문법 정리 시리즈 3편 추가(설계 단위/타입·프로세스/클럭·generate)

## [v0.0.11] - 2026-01-08

### Changed
- 모바일/태블릿 홈에서 프로필 아래 간단 카테고리 바 추가
- 모바일/태블릿에서 사이드바 카테고리 블럭 숨김(중복 제거)

## [v0.0.12] - 2026-01-08

### Changed
- 모바일/태블릿 홈 카테고리 UI를 접이식 패널로 변경(상위 클릭 → 하위 노출)
- 하위 카테고리 선택 시 해당 카테고리 페이지로 이동

## [v0.0.13] - 2026-01-08

### Fixed
- 모바일 카테고리 패널에서 하위 카테고리가 기본 노출되던 문제 수정(`hidden` 속성 우선 적용)

## [v0.0.14] - 2026-01-08

### Changed
- 모바일 카테고리 패널 토글 애니메이션 추가(하위 영역 높이/투명도/이동 전환)
- 버튼/칩 hover 및 active 상태 스타일 보강

## [v0.0.15] - 2026-01-08

### Added
- VHDL 문법 레퍼런스 단일 포스트 추가(문법 단위 아웃라인 중심)

## [v0.0.4] - 2026-01-08

### Changed
- 모바일 본문 가독성 개선(과도한 좌우 패딩 축소)
- dev 글(`post-container-wide`) 모바일 패딩 오버라이드 추가(실질 본문 폭 확장)

## [v0.0.2] - 2026-01-08

### Changed
- 모바일/태블릿 홈 상단 1줄 프로필 바 구성 변경(상태메시지 + 오늘 방문자수 표시)
- 방문자 통계 로더가 모바일 전용 요소도 함께 업데이트하도록 연결

### Fixed
- 모바일 상단바에서 오늘 방문자수 DOM 업데이트 누락 보완

## [2025-12-16] - UI/UX 개선 및 기능 추가

### 추가
- 24시간 이내 글에 "N" 배지 표시 기능
- 상대 시간 표시 (몇 초 전, 몇 분 전, 몇 시간 전 등)
- 포스트 카테고리 2단계 구조 (category > subcategory)
- 포스트 날짜에 시간 단위 추가

### 변경
- post-list-title 색상 및 폰트 크기 조정 (var(--text-secondary), 13px)
- post-list-title과 N 배지를 같은 줄에 표시하도록 구조 개선
- 인기글 섹션에서 "new" 배지 제거
- 방문자 통계 표시 형식 변경 ("T5" → "전체 5")
- 카테고리 경로 표시 로직 개선 (tags 대신 subcategory 사용)

### 수정
- 포스트 정렬 순서 수정 (최신글이 상단에 오도록)
- 포스트 날짜 형식에 타임존 추가 (+0900)
- CSS 스타일 최적화 및 정리

### 제거
- Vanta.js clouds 배경 효과 제거 (작동하지 않아 제거)

## [v0.1.0] - 초기 정리

- 상단 메뉴에 "버전" 항목 추가
- FPGA 플래시 메모리/BRAM 고찰 글을 dev 카테고리에 편성
- POSTS_OVERVIEW에서 카테고리 구조 정리
